FRONT-END
разработка

Курс "Front-End разработка" предоставляет участникам основы веб-разработки, охватывая HTML, CSS и JavaScript. Учащиеся научатся создавать отзывчивые и функциональные интерфейсы, адаптированные для разных устройств. В процессе обучения студентов ждут практические задания, направленные на развитие навыков в решении реальных задач. По окончании курса участники смогут создавать привлекательные и профессиональные веб-сайты, готовые к запуску на продакшен.

Критерии оценки

Успеваемость студентов оценивается по практическим заданиям, проектам и тестам.

Все задания и проекты оцениваются по критериям:

  • соответствие требованиям
  • качество кода
  • использование современных технологий

Максимальный балл за задание — 10.

Также предусмотрены тесты и экзамены, проверяющие знание теории и навыки разработки под iOS.

Формат обучения

  • Гибрид
  • Онлайн занятия проводятся в Google Meet / Zoom
  • Оффлайн занятия проводятся в учебной аудитории школы
  • Соотношение теории к практике - 50/50

Этапы и критерии отбора

Поступление на курс проходит в несколько этапов:

  1. Онлайн-анкета и эссе

    2. Студенты заполняют анкету и пишут короткое эссе, в котором рассказывают:

    • почему интересуются разработкой мобильных приложений.
    • почему хотят изучать iOS.
  2. Собеседование

    3. После отбора анкет мы приглашаем кандидатов на онлайн-собеседование.

Курс iOS-разработки — отличная возможность для тех, кто хочет начать карьеру в IT, научиться создавать современные приложения и получить ценные практические навыки.

    Мы рассматриваем только студентов от 18 лет и старше, которые:

  • заинтересованы в мобильной и web-разработке
  • обладают мотивацией к обучению
  • умеют или готовы учиться работать в команде
  • имеют базовые знания в программировании (желательно, но не обязательно).

Программа обучения

Неделя 1. Вступительное занятие

  • Обзор курса
  • Создание базовой HTML-страницы
  • HTML компоненты: ввод, кнопки, формы, элементы div, таблицы

Неделя 2. CSS. Введение.

  • Стилизация элементов
  • Классы
  • Стили, цвета, положение, интервалы, выравнивание, шрифты, классы
  • Стилизация шрифтов с помощью CSS и веб-безопасные шрифты.
  • Центрирование элементов с помощью CSS.
  • Селекторы CSS и приоритет селекторов.

Неделя 3. CSS (завершение)

  • Display
  • Flexbox
  • Grid
  • Игры на закрепление знаний

Неделя 4. Bootstrap. Введение.

  • Система компоновки сетки
  • Контейнеры, компоненты
  • Карусели Bootstrap
  • Карточки Bootstrap
  • Навигационные панели

Неделя 5. Tailwind CSS

Неделя 6. SASS/SCSS

  • @mixin, @include, @extend

Неделя 7. Анимация и переходы CSS

  • Animation
  • Transition
  • Keypoints

Неделя 8. Dark Mode. Темная тема.
